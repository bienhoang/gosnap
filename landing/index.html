Features

AI-ready feedback, built in

Every annotation captures CSS selectors, DOM paths, and metadata. Copy once, paste to any AI agent — no screenshots, no guessing.

AI-Ready Element Capture

Hover to highlight any element. Captures CSS selectors, DOM paths, accessibility data, and bounding boxes — so AI knows exactly which element you mean.

Ordered Annotations

Numbered step markers pinned to elements. AI follows your feedback in sequence. Markers survive scroll and resize.

AI-Optimized Output

Two modes: compact markdown or detailed with full DOM context, computed styles, and coordinates. Paste directly into any AI agent.

Keyboard Shortcuts

Full shortcut system — toggle, inspect, copy, navigate markers, undo delete. All from your keyboard.

Multi-Element Selection

Drag to select multiple elements and annotate as a group. Describe component-level issues to AI with one annotation.

Works Anywhere

~14 kB React component or ~76 kB standalone embed. Drop on any website — React, Next.js, Vue, HTML, WordPress. Zero config.

Quick Start

Annotate. Copy. Paste to AI.

Three steps to structured feedback that any AI agent can understand and act on.

1

Annotate — click elements, leave notes. Tool captures CSS selectors, DOM paths, and metadata.

2

Copy — press ⌘⇧C. Structured markdown copied to clipboard.

3

Paste to AI — AI knows exactly which elements, their properties, and what you want changed.

API

Fully customizable

Configure position, theme, z-index, persistence, and hook into every interaction.

Prop Type Default Description
position 'bottom-right' | 'bottom-left' | 'top-right' | 'top-left' 'bottom-right' Toolbar position on viewport
theme 'dark' | 'light' 'dark' Color theme
defaultCollapsed boolean true Start with toolbar collapsed
zIndex number 9999 Base z-index for all layers
persist boolean | string — Enable localStorage persistence with orphan detection
triggerIcon ReactNode <Menu /> Custom icon for trigger button

Callbacks

Prop Type Description
onToggle (active: boolean) => void Inspector started or stopped
onInspect (element: InspectedElement) => void Element selected via inspector
onFeedbackSubmit (feedback: FeedbackItem) => void Feedback annotation submitted
onFeedbackDelete (feedbackId: string) => void Feedback marker deleted
onFeedbackUpdate (feedbackId: string, content: string) => void Feedback text edited
onCopy () => void "Copy" toolbar button clicked
onDelete () => void "Delete All" toolbar button clicked
onFeedback () => void "Feedbacks" toolbar button clicked
onSettings () => void "Settings" toolbar button clicked

Shortcuts

Keyboard-first workflow

Every action has a shortcut. Navigate markers, toggle inspector, copy feedbacks, undo deletes — without touching the mouse.

⌘⇧F Toggle toolbar
⌘⇧I Toggle inspector
⌘⇧C Copy all feedbacks
⌘⇧L Open feedback list
⌘⇧, Open settings
⌘⇧⌫ Delete all
⌘Z Undo delete
[ ] Navigate markers
Enter Edit focused marker
Esc Close / dismiss

On Windows/Linux, use Ctrl instead of ⌘
